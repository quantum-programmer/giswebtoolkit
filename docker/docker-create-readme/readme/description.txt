  GIS WebToolKit Special Edition (SE) – инструментарий разработки
Web-приложений. Представляет собой набор JavaScript-скриптов
для встраивания компонентов доступа к пространственным данным
и отображения цифровой информации о местности. Инструментарий
обеспечивает получение атрибутивных характеристик объектов карты
по запросам, поиск объектов с заданными атрибутами или поиск
на основе пространственного фильтра. Выполняются расчеты
геометрических свойств объектов, топологические оверлейные операции
на векторных моделях пространственных объектов. Имеется набор программных
средств редактирования объектов карты, получения данных для
3D-моделирования.
  GIS WebToolKit SE поддерживает протоколы обмена пространственными
данными по стандартам Open Geospatial Consortium (OGC): Web Map Tile
Service (OGC WMTS), Web Map Service (OGC WMS), Web Feature Service
(OGC WFS), Web Coverage Service (OGC WCS) и дополнительные запросы
с использованием RESRT API. Запросы выполняются асинхронно по
протоколу HTTP с использованием технологии AJAX.
  Для проверки работы инструментария используется сервис GIS WebService SE,
расположенный по адресу https://gisserver.info/GISWebServiceSE/service.php

  Для использования инструментария необходимо настроить среду разработки,
установив в IDE менеджер пакетов npm командой npm install.
Структура каталога:
   GIS WebToolKit SE - скрипты карты, классов управления картой;
   src - скрипты компонентов карты на базе Vue.js и Vuetify, обертка карты для Vue.js;
   src/plugins/locale/*.json - файлы локализации компонентов инструментария;
   release - файлы сборки.

Для сборки библиотеки необходимо выполнить команду сборки.
Команды располагаются в файле package.json.

Результат размещается в папке release.

Для сборки библиотеки GWTKSE и Vue компонентов необходимо:
    в среде Linux: вызвать команду npm run build-lib
    в среде Windows: вызвать команду npm build-lib-win
В результате успешной сборки папка release будет содержать откомпилированные
библиотеки компонентов карты (с включенными текстами GWTKSE внутри) - файлы
MapVue.umd.min.js, MapVue.umd.min.[x].js., а также сопутствующие файлы.

Для сборки только ядра GWTKSE необходимо вызвать команду npm run build-gwtk.
В результате успешной сборки папка release будет содержать откомпилированные
скрипты карты, классов управления картой - файл gwtkse.js, файл стилей
gwtkse.css, а также сопутствующие файлы.
