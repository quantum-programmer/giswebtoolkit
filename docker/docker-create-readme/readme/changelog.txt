27.07.23 Версия 6.15.0
-------------------------------------------------------------------

 В новой версии расширены инструменты пространственного анализа и поиска
данных. Добавлены компоненты построения зоны затопления и поиска
объектов по названию, доработан модуль администрирования.
Компонент построения зоны затопления позволяет определить область
затопления на карте, используя информацию из матрицы высот. При
обращении к серверу приложений GIS WebService SE исследуется уклон
анализируемого участка местности, формируются створы по руслу реки
и учитывается уровень подъема воды. Результат выводится в виде
изображения матрицы качеств и векторного объекта, показывающего
границы разлива.
Компонент Поиск по названию используется для текстового поиска объектов
по всем характеристикам. Условия могут быть конкретизированы отдельно
для каждой из карт, также имеется возможность выполнять групповой
запрос по всем картам. Параметры поиска могут быть восстановлены из
хранимой истории запросов.

16.06.23 Версия 6.14.0
-------------------------------------------------------------------

 В новой версии расширены средства пространственного анализа и поиска
данных. Обновлен компонент Списки объектов, доработан поиск в
компоненте Состав карты. В компонент Списки объектов добавлен поиск
объектов по пересечению контуров. Результаты поиска
позволяют анализировать связи между различными типами объектов по их
расположению на местности. Списки объектов представляют собой
выборки пространственных объектов карты. Проверка пересечения
контуров выполняется между объектами двух списков в соответствии с
выбранным критерием пересечения. Например, можно найти объекты первого
списка, имеющие точку касания, или расположенные внутри объектов второго
списка и не имеющие общих точек, или расположенные снаружи, или не
пересекающиеся. Отобранные объекты выделяются в карте, их описание можно
просмотреть и скачать. Для поиска пересечений по спискам объектов
необходимо использовать сервис GIS WebService SE версии не ниже 15.2.5.
Обновленный компонент Состав карты предоставляет возможность быстрого
поиска слоев карты по тэгам. Для поиска по тегам необходимо выбрать
хотя бы один тег из списка или ввести значение в строке поиска.
Тэги слоев описываются в параметрах карты.

Поддерживается редактирование загруженного слоя. Результаты редактирования могут быть
сохранены на компьютере клиента.
  Добавлена возможность получения списка объектов для отдельного слоя. Список
формируется в табличном виде и содержит все семантики объектов. Доступны средства
сортировки и фильтрации объектов. При выборе объекта в списке, происходит
его поиск на карте. Данный режим доступен в панелях "Состав карты" и "Списки
объектов".
  Добавлен компонент "Поиск по семантике". Компонент предназначен для поиска объектов
карты по семантическим данным или по идентификационному номеру объекта.
  Добавлен компонент "Параметры карты". Компонент позоляет изменять прозрачность слоя
и порядок отображения слоев карты.
